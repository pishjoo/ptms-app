# PTMS

Professional Trade Monitoring System.

## Overview

PTMS is a foreign trade process monitoring platform focused on trade-case visibility, monitoring workflows, and operational coordination.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma ORM
- PostgreSQL-ready configuration

## Development

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env.local` and configure your PostgreSQL connection string.
